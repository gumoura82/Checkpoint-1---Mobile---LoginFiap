# Projeto: Tela de Login FIAP

## Descrição do Projeto
Este projeto consiste no desenvolvimento de uma tela de login inspirada no site da FIAP, utilizando React Native. A tela foi criada com foco visual nas cores preta e vermelha, representando a identidade visual da faculdade FIAP. O objetivo é demonstrar a construção de interfaces mobile modernas, utilizando componentes personalizados e estilização adequada, sem implementar a funcionalidade de autenticação.

### Ideia Principal
- Criar uma interface de login semelhante à do site da FIAP, priorizando a identidade visual da instituição.
- Utilizar componentes base do React Native (Texto, Input, Botão, etc.) e componentes personalizados.

### Requisitos
- Node.js (versão recomendada: 16.x ou superior)
- npm ou yarn
- Expo CLI instalado globalmente (`npm install -g expo-cli`)
- Emulador Android/iOS ou dispositivo físico para testes

#### Como rodar o projeto
1. Clone este repositório
2. Instale as dependências com `npm install` ou `yarn`
3. Inicie o projeto com `expo start`
4. Escaneie o QR code com o aplicativo Expo Go ou rode no emulador
